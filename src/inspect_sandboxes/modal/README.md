# Modal Sandbox

Serverless container sandbox for [Inspect AI](https://inspect.aisi.org.uk/) using [Modal](https://modal.com).

For setup, configuration, and usage, see the [Modal provider page](https://meridianlabs-ai.github.io/inspect_sandboxes/modal.html) on the [`inspect_sandboxes` documentation site](https://meridianlabs-ai.github.io/inspect_sandboxes/).

Based on [@anthonyduong9](https://github.com/anthonyduong9)'s work.
