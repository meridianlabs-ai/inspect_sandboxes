# Daytona Sandbox

Cloud development environment sandbox for [Inspect AI](https://inspect.aisi.org.uk/) using [Daytona](https://www.daytona.io/).

For setup, configuration, and usage, see the [Daytona provider page](https://meridianlabs-ai.github.io/inspect_sandboxes/daytona.html) on the [`inspect_sandboxes` documentation site](https://meridianlabs-ai.github.io/inspect_sandboxes/).
